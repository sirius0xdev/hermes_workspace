Built on battle-tested infrastructure

Google Kubernetes Engine, Cilium networking, RTX 6000 Pro GPUs, and spot+on-demand resilience.

🏢 Worker Nodes

All user workloads—including agents, Postgres, web hosting, and management—run on dedicated GKE worker nodes. Nothing runs on the Kubernetes control plane.

🌈 GPU Node Pools

Spot primary + On-Demand fallback with node selectors/taints. Graceful preemption handling. RTX 6000 Pro Ada GPUs.

🔌 Networking

Cilium Cluster Mesh for seamless communication between clusters and services. GatewayAPI for host-based routing.

⚡ Scaling

On-demand GPU spin-up for inference. Auto-shutdown when idle. KEDA scaling for intelligent resource management.
